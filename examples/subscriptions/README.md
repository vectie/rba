# Subscription Example

This example is a minimal debugging page for Rabbita subscriptions.

It covers the non-animation subscriptions:

- `every`
- `on_resize`
- `on_scroll`
- `on_key_down`
- `on_key_up`
- `on_visibility_change`
- `on_mouse_move`

The `on_animation_frame` demo now lives in `examples/animation`.

## Run in development

In this directory:

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite (usually `http://localhost:5173`).

## Build

```bash
npm run build
```

The built assets are generated in `dist/`.
